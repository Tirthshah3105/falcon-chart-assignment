# Falcon Chart Assignment

This project contains reusable Angular chart components.

Implemented:
- Column Chart
- Line Chart
- Dynamic Data Binding
- Scalable Values
- Responsive Layout
Developed using Angular Standalone Components.

## How to Run

npm install
ng serve
